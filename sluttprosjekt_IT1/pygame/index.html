PYGAME

Flere er kanskje kjent med python fra matten. Pygame er en morsom del av Python språket som er veldig egnet til å lage enkle, men kule og morsomme spill. På denne siden er det litt om pygame og mine pygame prosjekter fra i år.

Hva er pygame

Pygame er et bibliotek fra Python. Biblioteket inneholder masse innebygde funksjoner som går det lettere å lage visuelle ting. Dette kan være spill, animasjoner, illustrasjoner... egenlig hva som helst visuelt. Prosjektene kan være store, men også veldig småe og enkle.

Mine pygame prosjekter

Jumpy

Hoppe spill om å hoppe oppover så høyt som mulig er veldig populære. Jeg har laget mitt eget, toplayer spill med noen egne elemnter. Mitt går ut på å hoppe så høyt som mulig på 40 sek og å slå motstanderen. Faller du av, taper du. Treffer du motstanderene, taper du. Er du lavest når tiden er ute, taper du.
